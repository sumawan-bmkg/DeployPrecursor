## Analysis Sequence

### Step 1: Load Data
```python
# Input files:
prediction_registry = "Campaign/Blind_Test/prediction_registry.csv"
ground_truth = "initial/merge2026.csv"
matched_results = "Campaign/Blind_Test/matched_results.csv"  # generated by matching script

# Verify hashes before loading
assert sha256(prediction_registry) == MANIFEST_HASH
```

### Step 2: Confusion Matrix
- Compute TP, FP, FN, TN at confidence threshold = optimal threshold
- Generate 2×2 confusion matrix heatmap
- Report: precision, recall, specificity, F1, balanced accuracy, MCC

### Step 3: ROC Curve
- Sweep threshold 0.01-0.99
- Compute TPR/FPR at each step
- Calculate AUC with trapezoidal integration
- Bootstrap 95% CI (5,000 resamples)
- Plot: ROC curve with CI band + random diagonal

### Step 4: PR Curve
- Sweep threshold 0.01-0.99
- Compute precision/recall at each step
- Calculate AUC-PR (average precision)
- Compare to baseline AUC-PR = prevalence
- Plot: PR curve + baseline line

### Step 5: Calibration
- Bin predictions into 10 equal-width bins by confidence
- Compute mean confidence and observed accuracy per bin
- Calculate ECE and MCE
- Plot: reliability diagram + perfect calibration diagonal

### Step 6: Bootstrap CIs
```python
for metric in [precision, recall, F1, AUC, MCC]:
    for b in 1:5000:
        sample = resample(matched_results)
        metric_b = compute(sample)
    ci = percentile([2.5, 97.5])
```

### Step 7: Permutation Test
```python
true_f1 = compute_f1(matched_results)
shuffled_f1s = []
for p in 1:1000:
    shuffled = shuffle_event_times(matched_results)
    shuffled_f1s.append(compute_f1(shuffled))
```

### Step 8: Null Model Comparison
- Generate random baseline (same prevalence)
- Generate persistence baseline
- Generate threshold baseline (H-var > 3σ)
- Run Isolation Forest, One-Class SVM, CNN-1D, physics-only rule
- Compare all via McNemar's test and paired bootstrap

### Step 9: Sensitivity Analysis
- Sweep confidence threshold (0.1-0.9)
- Sweep tolerance window (6h, 12h, 24h, 48h)
- Sweep magnitude cutoff (3.5, 4.0, 4.5, 5.0)
- Sweep spatial tolerance (100, 300, 500, 1000 km)
- Report: surface plots, min/max F1, stability assessment

### Step 10: Per-Station Metrics
- Compute all metrics individually for each of 21 stations
- Apply Benjamini-Hochberg FDR correction
- Report: station performance table, station map heatmap

### Step 11: Stratified Analysis
- By Dst range (quiet/mod/storm/severe)
- By time of day (day/night)
- By EEJ zone (core/transition/non-EEJ)
- By magnitude range

### Step 12: Lead Time Distribution
- Histogram of lead times for all TPs
- Mean, median, std, IQR
- Bootstrap CI for mean lead time
- Lead time vs magnitude scatter plot

### Step 13: Physics Validation
- Z/H ratio during TP vs quiet periods (Wilcoxon test)
- Polarization ratio comparison
- Dst correlation with prediction confidence (Pearson r)
